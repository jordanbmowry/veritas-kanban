Build a custom Ghost theme for digitalmeld.io inspired by the Handshake Influence Dashboard design by Fuselab Creative / George Railean.

Design reference: obsidian://open?vault=Documents&amp;file=research/handshake-dashboard-design-analysis

Key design principles:

- Dark layered backgrounds (elevation via shade, not shadows)
- 3-level text hierarchy (white headlines → medium body → small muted labels)
- Inter font family + monospace for stats
- 8px spacing grid, generous padding
- Near-invisible borders, spacing-based separation
- Single blue accent color for CTAs/active states
- Pill tabs, stat cards, ranking lists
- Zero drop shadows

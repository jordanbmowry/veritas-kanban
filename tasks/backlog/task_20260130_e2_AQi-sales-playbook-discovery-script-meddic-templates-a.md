**Goal:** Create the team's operational sales playbook — the document every team member uses in live sales conversations.
**Owner:** Brad (author) + Mark (field testing)
**Effort:** M (8-10 hours)

**Acceptance Criteria:**

- Discovery call script documented (Opening → MEDDIC Questions → Budget Discussion → Closing → Post-Call)
- MEDDIC scoring template ready (0-3 per element, 18 max, qualify/disqualify thresholds)
- Objection handling guide with responses for top 10 objections
- Pre-call questionnaire template in HubSpot
- Post-call follow-up email template in HubSpot
- Pricing presentation approach documented (Value First, Tiered Options, ROI Calculator)

**Subtasks:**

1. Finalize discovery call script — 5-section structure (Opening 5min, Discovery 20-30min with MEDDIC questions, Budget 5min, Closing 5min, Post-Call within 1 hour) — print for each team member (2h) — Ref: HubSpot Guide §9, Discovery Call Script
2. Create MEDDIC scoring template — 6 elements scored 0-3, thresholds defined (15+ highly qualified, 10-14 proceed with caution, &lt;10 disqualify/nurture) — create as Google Sheet or HubSpot note template (2h) — Ref: HubSpot Guide §9, MEDDIC Breakdown + Scoring
3. Write objection handling guide — responses for: Price Too High, Need to Think, Working with Vendor, Building In-House, No Budget, Not Ready, Need Boss Approval, Send Proposal, How Long, Why You vs Competitor (2h) — Ref: HubSpot Guide §9, Objection Handling (all 10)
4. Create pricing presentation materials — Value-First framework, 3-tier pricing template (Essential/Recommended/Premium), ROI calculator spreadsheet (2h) — Ref: HubSpot Guide §9, Pricing Presentation Approach
5. Build email templates in HubSpot: Pre-Call Questionnaire, Post-Discovery Follow-Up, Proposal Follow-Up (Day 3), Stale Deal Nudge (Day 14), Won Deal Welcome, Lost Deal Thank You (2h) — Ref: HubSpot Guide Appendix, Email Templates
6. Team role-play session: practice discovery script + objection handling with Mark, Robert, Brandy (2h)
